# 🕯️ Tienda de Velas - E-commerce

## 🚀 Tecnologías
- React + Vite
- Supabase (Auth, DB, RLS)

## ⚙️ Configuración local
1. Clonar repo
2. Instalar dependencias `npm install`
3. Crear `.env` con:
   VITE_SUPABASE_URL=
   VITE_SUPABASE_ANON_KEY=
4. Ejecutar `npm run dev`
